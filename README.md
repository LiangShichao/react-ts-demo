# React typescript demo

## Prepare required
    - Node.js & NPM
    - React
    - React-admin
    - Webpack
    - JavaScript & HTML
    - TypeScript

## Install develope env
    - Visual Studio Code
    - install dependencies as bellows:
``` shell
$ npm install -g webpack
$ npm install
$ npm start
```
    - visit brower on http://localhost:3000
